# Data Extraction API

## Description
API to extract text from PDF and DOCX documents.

## Tech Stack
- Python
- Flask
- pdfplumber
- python-docx

## Setup Instructions
1. Install dependencies:
   pip install -r requirements.txt
2. Run:
   python app.py

## API Endpoint
POST /analyze-document

## Authentication
Header:
x-api-key: 12345
